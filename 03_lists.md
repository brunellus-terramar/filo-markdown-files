### Recipe: Simple Pasta

1. Boil a large pot of salted water
2. Add pasta and cook according to package directions
3. While pasta cooks, prepare your sauce
4. Drain pasta, reserving 1 cup of pasta water
5. Combine pasta with sauce
6. Add pasta water as needed for consistency
7. Serve immediately with grated cheese

## Inline Formatting in Lists

Testing bold, italic, and inline code within list items:

- This item has **bold text** in it
- This item has *italic text* in it
- This item has `inline code` in it
- This item has ***bold and italic*** text
- Combining **bold**, *italic*, and `code` in one item
- Here's a function call: `console.log("hello world")`

Ordered list with formatting:

1. First step: **Create a new file**
2. Second step: Run `npm install` to install dependencies
3. Third step: Add *configuration* settings
