# Welcome to Markdown Reader

This is a sample document to demonstrate basic markdown rendering.

## Introduction

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents. Created by John Gruber in 2004, Markdown is now one of the world's most popular markup languages.

## Why Use Markdown?

There are several reasons why you might want to use Markdown:

- **Portability**: Markdown files are plain text, so they can be opened by any application.
- **Platform Independence**: You can create Markdown files on any operating system.
- **Future Proof**: Even if the application you're using stops working, you can still read your Markdown files.
- **Widespread Support**: Many websites and applications support Markdown.

## Basic Formatting

You can make text **bold** or *italic* easily. You can also combine them for ***bold and italic*** text.
